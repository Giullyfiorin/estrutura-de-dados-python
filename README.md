# Estrutura de Dados em Python

Este projeto foi desenvolvido como parte da disciplina de Estrutura de Dados, com o objetivo de aplicar conceitos fundamentais na prática.

## 📌 Conteúdos implementados

### 🔹 Lista Encadeada (Fila com Prioridade)

Sistema de triagem hospitalar onde:

* Pacientes com prioridade (A) são atendidos antes
* Pacientes sem prioridade (V) entram no final da fila
* Numeração automática dos atendimentos

Funcionalidades:

* Inserção com e sem prioridade
* Remoção (atendimento)
* Exibição da fila

---

### 🔹 Tabela Hash com Encadeamento

Implementação de tabela hash com:

* 10 posições
* Tratamento de colisão com lista encadeada
* Função hash baseada em ASCII
* Regra especial para DF

Funcionalidades:

* Inserção de estados brasileiros
* Inserção de elemento fictício
* Impressão da tabela

---
